# Server

`AWS`를 사용하여 지속적으로 자신의 블로그를 `감시`하고, 컨트리뷰션 그래프에 삽입될 `데이터를 생성`합니다. 생성된 데이터는 `Tistory API`를 사용하여 특정 게시글에 저장/갱신됩니다.

---

### 사용된 스택

-   `NodeJS With Typescript`
-   `Serverless`
-   `AWS Lambda`
-   `AWS CloudWatch`
